# bnb-ui

# Initialization

- Step 1: Install NodeJS, yarn
- Step 2: Run commnad:
  yarn
  yarn dev

# To run test with Cypress

- Step 1: Run command:
  npx cypress open

- Step 2: Choose `E2E Testing` option

- Step 3: Choose your `browser` and `Start E2E Testing in {browser}`

- Step 4: Choose the test section you want to run
